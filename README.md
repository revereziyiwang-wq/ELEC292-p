# ELEC292 Project - Data Analysis

Welcome to the repository for ELEC292. This project contains raw data and processing scripts in Python.

## 📂 Project Structure
- **/Rawdata**: Original, untouched data files.
- **/Data**: Processed datasets.
- **main.py**: The primary script for running our analysis.

---

## 🛠 How to Work on This Project
The `main` branch is **protected**. Please follow these steps to add your code:

### 1. Update your local files
Before starting, always get the latest version:
`git pull origin main`

### 2. Create a new branch
Never work directly on main! Create a branch for your task:
`git checkout -b yourname-task-description`

### 3. Save and Push
When you finish a feature or fix, run these three commands in order to send your changes to GitHub:
Bash
git add .                           # 1. Stage all your changes
git commit -m "Brief description"   # 2. Save changes locally
git push                            # 3. Send to GitHub
